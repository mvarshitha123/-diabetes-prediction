# Diabetes Prediction using Machine Learning

## 📌 Overview
This project predicts whether a person has diabetes using Machine Learning (Logistic Regression).

## 🎯 Goal
To analyze health data and predict diabetes based on features like glucose, BMI, blood pressure, and age.

## 🤖 Algorithm Used
- Logistic Regression

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## 📊 Features
- Data preprocessing
- Model training
- Prediction system
- Accuracy evaluation

## 🚀 Output
The model predicts:
- 0 → No Diabetes
- 1 → Diabetes Detected
